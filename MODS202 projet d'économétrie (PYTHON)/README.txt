# Projet d'analyse économétrique

Ce projet vise à réaliser une analyse économétrique sur un jeu de données des trois derniers siècles, explorant diverses relations entre les variables économiques.

## Description du Projet

Ce projet utilise des techniques économétriques pour analyser les relations entre différentes variables économiques telles que le PIB, le taux de chômage et l'inflation. Il comprend plusieurs étapes :

- Nettoyage et prétraitement des données
- Analyse exploratoire des données
- Modélisation économétrique
- Tests de causalité et de stabilité des coefficients
- Détection de ruptures structurelles

## Structure des Fichiers

Le projet est structuré comme suit :

- `data/` : Répertoire contenant les données brutes
- `projet_econométrie_final_Manel_Hamza/` : Scripts Python utilisés pour l'analyse
- `README.md` : Fichier actuel, fournissant une documentation et des instructions

## Exécution du Code

1. Assurez-vous d'avoir Python installé sur votre machine.
2. Clonez ce dépôt localement.
3. Exécutez les scripts Python dans l'ordre numéroté pour reproduire l'analyse.

## Dépendances

Ce projet nécessite l'installation de certaines bibliothèques Python. Consultez le fichier pour obtenir la liste complète des dépendances.
