# Project_Labyrinthe_Hamza_El_Arji_JAVA

## Description

Ce projet Java est un petit jeu interactif permettant aux utilisateurs de résoudre des labyrinthes de taille n*n et hexagonaux, de modifier et de supprimer des murs du labyrinthe, ainsi que de rajouter une case de départ et d'arrivée.


## Installation

1. Clonez le dépôt GitHub : `https://gitlab.telecom-paris.fr/2022inf103/groupe1/el-arji-hamza.git`

2. Importez le projet dans votre environnement de développement Java

3. Exécutez la classe principale "Main.java" dans le package uiView pour lancer l'application


## Utilisation

Lorsque vous lancez l'application, vous verrez une interface graphique simple avec des boutons pour ajouter, modifier et supprimer des cases. Par défaut un labyrinthe s'affiche. Vous pouvez importer (de taille n*n) ou exporter des labyrinthes.
